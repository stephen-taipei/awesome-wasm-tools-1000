## CMP-001 ZIP 壓縮

| 欄位 | 內容 |
|------|------|
| **用途** | 將檔案或資料夾壓縮為 ZIP 格式 |
| **技術重點** | Deflate 壓縮、壓縮等級、目錄結構保留 |
| **WASM 模組** | zip-wasm (~800 KB) |
| **執行流程** | 1. 選擇檔案 → 2. 設定壓縮等級 → 3. 壓縮 → 4. 下載 |
| **輸入格式** | 任意檔案/資料夾 |
| **輸出格式** | ZIP |
| **效能需求** | 記憶體：檔案大小 x2、處理時間：依大小 |
| **進度狀態** | ⬜ 待開發 |
