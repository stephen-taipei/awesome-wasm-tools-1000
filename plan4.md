## CRY-001 AES 加密

| 欄位 | 內容 |
|------|------|
| **用途** | 使用 AES 演算法加密檔案或文字 |
| **技術重點** | AES-128/192/256、CBC/CTR/GCM 模式、IV 生成 |
| **WASM 模組** | crypto-wasm (~500 KB) |
| **執行流程** | 1. 輸入資料 → 2. 設定密鑰 → 3. 選擇模式 → 4. 加密 → 5. 下載 |
| **輸入格式** | 文字、任意檔案 |
| **輸出格式** | 加密檔案、Base64 |
| **效能需求** | 記憶體：檔案大小 x2、處理時間：快速 |
