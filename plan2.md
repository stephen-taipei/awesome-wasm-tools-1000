## AUD-001 MP3 轉 WAV

| 欄位 | 內容 |
|------|------|
| **用途** | 將 MP3 音訊轉換為無損 WAV 格式 |
| **技術重點** | MP3 解碼、PCM 輸出、取樣率保留 |
| **WASM 模組** | ffmpeg-wasm (~25 MB) |
| **執行流程** | 1. 上傳 MP3 → 2. 解碼 → 3. 輸出 WAV → 4. 下載 |
| **輸入格式** | MP3 |
| **輸出格式** | WAV |
| **效能需求** | 記憶體：音訊大小 x3、處理時間：1-5秒 |
| **進度狀態** | ⬜ 待開發 |
