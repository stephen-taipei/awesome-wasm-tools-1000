📋 TIFF 格式說明

TIFF (Tagged Image File Format) 是專業圖片格式，特點如下：

  • 🔷 無損壓縮：保留完整畫質，適合專業編輯
  • 🔷 多頁支援：單一檔案可包含多個頁面/圖層
  • 🔷 色彩深度：支援 8/16/32 位元色彩
  • 🔷 色彩空間：支援 RGB、CMYK、Lab 等
  • 🔷 常見用途：印刷、掃描、攝影後製

💡 瀏覽器對 TIFF 的原生支援有限，建議轉換為 PNG 以便網頁使用
