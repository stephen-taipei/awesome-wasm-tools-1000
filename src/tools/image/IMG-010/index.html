📋 SVG 格式說明

SVG (Scalable Vector Graphics) 是向量圖格式，特點如下：

  • 🔷 無限縮放：放大不失真，適合各種解析度
  • 🔷 檔案小：適合簡單圖形和圖示
  • 🔷 可編輯：XML 格式，可用文字編輯器修改
  • 🔷 網頁優先：現代瀏覽器原生支援

💡 轉換為 PNG 後將成為點陣圖，建議設定適當的輸出尺寸
