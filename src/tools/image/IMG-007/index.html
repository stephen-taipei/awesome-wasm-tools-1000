📋 BMP 格式說明

BMP (Bitmap) 是 Windows 系統的標準圖片格式，特點如下：

  • 🔷 無壓縮：檔案較大但保留完整畫質
  • 🔷 廣泛相容：所有 Windows 應用程式都支援
  • 🔷 位元深度：支援 1/4/8/16/24/32 位元
  • 🔷 透明度：32 位元 BMP 支援 Alpha 通道

💡 建議轉換為 PNG 或 WebP 以獲得更小的檔案大小

所有檔案處理均在本地完成，不會上傳到伺服器
