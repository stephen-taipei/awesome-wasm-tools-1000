📋 ICO 格式說明

ICO 是 Windows 系統使用的圖示格式，特點如下：

  • 🔷 多尺寸：單一檔案包含多種解析度圖示
  • 🔷 常見尺寸：16, 32, 48, 64, 128, 256 像素
  • 🔷 透明支援：支援 Alpha 通道透明度
  • 🔷 用途：桌面圖示、網站 favicon、應用程式圖示

💡 建議使用正方形 PNG 圖片（含透明背景）以獲得最佳效果

所有檔案處理均在本地完成，不會上傳到伺服器
