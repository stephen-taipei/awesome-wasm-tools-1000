## IMG-001 PNG 轉 JPG

| 欄位 | 內容 |
|------|------|
| **用途** | 將 PNG 圖片轉換為 JPG 格式，支援品質調整 |
| **技術重點** | Canvas API 繪製 + toBlob 輸出、品質參數控制 |
| **WASM 模組** | 原生 Canvas API (無需 WASM) |
| **執行流程** | 1. 讀取 PNG → 2. 繪製到 Canvas → 3. 輸出 JPG |
| **輸入格式** | PNG |
| **輸出格式** | JPG/JPEG |
| **效能需求** | 記憶體：圖片大小 x3、處理時間：<1秒/張 |
| **進度狀態** | ✅ 已完成 |
