# Plan 1-8: 圖片處理工具 (IMG-141 ~ IMG-150)

> 共 150 個工具，本檔案包含 IMG-141 ~ IMG-150（圖片處理最後 10 個工具）

## IMG-141 HDR 合成

| 欄位 | 內容 |
|------|------|
| **用途** | 將多張不同曝光的照片合成為 HDR 圖片 |
| **技術重點** | 曝光融合、色調映射、對齊校正 |
| **WASM 模組** | photon-wasm (~2 MB) |
| **執行流程** | 1. 上傳多張不同曝光照片 → 2. 自動對齊 → 3. HDR 合成 → 4. 下載 |
| **輸入格式** | JPG, PNG |
| **輸出格式** | JPG, PNG |
| **效能需求** | 記憶體：所有圖片總和 x3、處理時間：5-15秒 |
| **進度狀態** | ⬜ 待開發 |
