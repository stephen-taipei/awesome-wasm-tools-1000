## IMG-121 JXL 格式轉換

| 欄位 | 內容 |
|------|------|
| **用途** | JPEG XL 格式轉換（次世代圖片格式） |
| **技術重點** | JXL 編解碼、無損/有損壓縮、漸進式載入 |
| **WASM 模組** | jxl-wasm (~2 MB) |
| **執行流程** | 1. 上傳圖片 → 2. 選擇轉換方向 → 3. 轉換 → 4. 下載 |
| **輸入格式** | PNG, JPG, JXL |
| **輸出格式** | JXL, PNG, JPG |
| **效能需求** | 記憶體：圖片大小 x3、處理時間：2-5秒 |
| **進度狀態** | ⬜ 待開發 |
