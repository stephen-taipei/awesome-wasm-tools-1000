## CMP-041 檔案合併

| 欄位 | 內容 |
|------|------|
| **用途** | 將分割的檔案合併還原 |
| **技術重點** | 順序偵測、完整性校驗、自動命名 |
| **WASM 模組** | 原生 File API (無需 WASM) |
| **執行流程** | 1. 上傳分割檔 → 2. 自動排序 → 3. 合併 → 4. 下載 |
| **輸入格式** | 分割檔案 (.001, .002...) |
| **輸出格式** | 原始檔案 |
| **效能需求** | 記憶體：合併後大小、處理時間：依大小 |
| **進度狀態** | ⬜ 待開發 |
