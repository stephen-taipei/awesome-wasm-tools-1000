## CMP-101 檔案類型轉換

| 欄位 | 內容 |
|------|------|
| **用途** | 修正錯誤副檔名或轉換檔案類型 |
| **技術重點** | Magic Number 偵測、副檔名校正 |
| **WASM 模組** | 原生 (無需 WASM) |
| **執行流程** | 1. 上傳檔案 → 2. 偵測實際格式 → 3. 修正/轉換 → 4. 下載 |
| **輸入格式** | 任意檔案 |
| **輸出格式** | 正確副檔名檔案 |
| **效能需求** | 記憶體：檔案大小、處理時間：即時 |
| **進度狀態** | ⬜ 待開發 |
