## TXT-141 OCR 文字辨識

| 欄位 | 內容 |
|------|------|
| **用途** | 從圖片中辨識文字 |
| **技術重點** | Tesseract、多語言支援、區域選擇 |
| **WASM 模組** | tesseract-wasm (~15 MB) |
| **執行流程** | 1. 上傳圖片 → 2. 選擇語言 → 3. 辨識 → 4. 輸出文字 |
| **輸入格式** | 圖片 (PNG/JPG) |
| **輸出格式** | 辨識文字 |
| **效能需求** | 記憶體：高、處理時間：依圖片大小 |
| **進度狀態** | ⬜ 待開發 |
