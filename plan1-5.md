## IMG-081 噪點添加

| 欄位 | 內容 |
|------|------|
| **用途** | 為圖片添加顆粒噪點效果（底片感） |
| **技術重點** | 高斯噪點、均勻噪點、強度控制 |
| **WASM 模組** | photon-wasm (~2 MB) |
| **執行流程** | 1. 上傳圖片 → 2. 選擇噪點類型 → 3. 調整強度 → 4. 輸出 |
| **輸入格式** | PNG, JPG, WebP |
| **輸出格式** | PNG, JPG |
| **效能需求** | 記憶體：圖片大小 x2、處理時間：<1秒 |
