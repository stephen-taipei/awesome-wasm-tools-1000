## IMG-101 圖片轉 Base64

| 欄位 | 內容 |
|------|------|
| **用途** | 將圖片轉換為 Base64 編碼字串 |
| **技術重點** | FileReader API、DataURL 生成、格式選項 |
| **WASM 模組** | 原生 File API (無需 WASM) |
| **執行流程** | 1. 上傳圖片 → 2. 編碼為 Base64 → 3. 複製/下載 |
| **輸入格式** | PNG, JPG, WebP, GIF |
| **輸出格式** | Base64 字串, DataURL |
| **效能需求** | 記憶體：圖片大小 x2、處理時間：即時 |
